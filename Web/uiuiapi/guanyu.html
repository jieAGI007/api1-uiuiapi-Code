📚 UIUI API开发接入教程

开发文档

主站接口地址

https://api1.uiuiapi.com

修改应用 BASE_URL

将【api.openai.com】修改为以下中转接口调用地址：

https://api1.uiuiapi.com
https://api1.uiuiapi.com/v1
https://api1.uiuiapi.com/v1/chat/completions

点击查看接入文档，不同客户端BASE_URL，请尝试以下地址：

方法一 (Python):

import openai

openai.api_base = "https://api1.uiuiapi.com/v1"

方法二 (环境变量):

修改环境变量OPENAI_API_BASE，各个系统修改环境变量方法请自行搜索，修改后需重启系统。

OPENAI_API_BASE=https://api1.uiuiapi.com/v1

🎨 Midjourney接入

支持直接在聊天中使用。

MJ代理接口地址

URL: https://api1.uiuiapi.com

端口: 443 (如无可忽略不填)

密钥令牌Secret: 自己后台生成的令牌

支持最新Midjourney Proxy Plus接口协议

指令A组价格：mj_imagine (绘图)、mj_variation (变换)、mj_reroll (重绘)、mj_blend (混合)、mj_modal (窗口提交，局部重绘和自定义比例变焦必须和mj_modal一同添加)、mj_zoom (比例变焦)、mj_shorten (提示词缩短)、mj_high_variation (强变换)、mj_low_variation (弱变换)、mj_pan (平移)指令固定0.2/次

指令B组价格：mj_describe (图生文)、mj_upscale (放大)、swap_face (换脸)指令固定0.1/次

其中mj_inpaint (局部重绘提交，必须和mj_modal一同添加)和mj_custom_zoom (自定义比例变焦，必须和mj_modal一同添加)的价格设置为0，是因为这两个模型需要搭配mj_modal使用，所以价格由mj_modal决定。

上面为MJ代理的快捷接入方式，如果你的项目不支持以上方式，请点击查看“接入文档”

如果您需要以chat格式进行调用，请直接使用模型名称：mj-chat
